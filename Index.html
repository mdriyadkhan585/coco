Introduction

Welcome to the official documentation for **Coco**, the most powerful and advanced programming language created by **RK Riad** of **RK Studio**. Coco is designed to deliver exceptional performance, safety, and simplicity. Whether you’re a seasoned developer or new to programming, Coco provides the tools and features you need to build robust and efficient applications.

Coco combines the best aspects of C/C++, Python, Rust, and Ruby, offering a compelling mix of speed, safety, and ease of use.

Installation

To get started with Coco, follow these installation steps: 

                
# Install Coco using the official package manager
$ coco install coco-lang

Once installed, you can verify the installation by checking the version: 

                
$ coco --version

Usage

Here’s a quick example to help you get started with Coco: 

                
# main.co
fn main() {
    let name = "World"
    print("Hello, ", name, "!")
}

To run your Coco program, use the following command: 

                
$ coco run main.co

Features

  • Blazing Fast Performance: Achieve speeds comparable to C/C++ with dynamic optimizations.
  • Memory Safety: Advanced memory management prevents common errors.
  • Simple and Powerful Syntax: Combines intuitive syntax with powerful features.
  • Concurrency: Easy-to-use concurrency models for multi-core processors.
  • Type Inference: Enjoy static typing benefits with less verbosity.
  • Extensive Standard Library: Includes modules for various functionalities.
  • IDE Integration: Support for modern IDEs with features like code completion and debugging.
  • Interoperability: Easily interface with existing C/C++ codebases.

Examples

Concurrency Example

                
fn main() {
    let mut handles = []
    for i in 1..=10 {
        handles.push(spawn(|| {
            print("Task ", i, " running.")
        }))
    }

    for handle in handles {
        handle.join()
    }
    print("All tasks completed.")
}

Interfacing with C/C++

                
extern "C" {
    fn printf(format: *const i8, ...) -> i32;
}

fn main() {
    let msg = "Hello from C!"
    unsafe {
        printf("%s\n", msg)
    }
}
